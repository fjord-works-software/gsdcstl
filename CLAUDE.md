## Project Overview

This is a static HTML website for the German Shepherd Dog Club of St. Louis, created using Artisteer v4.3.0. The site consists of individual HTML files without build tools or automated workflows.

## Architecture

The site uses a simple file-based structure:
- Main pages in `gsdcstl.org/` root directory (`.html`, `.css`, `.js`)
- Subdirectories for page-specific content (e.g., `club-info/`, `events/`, `photo-gallery/`)
- Assets in `gsdcstl.org/assets/` (images, documents)

All pages are self-contained HTML files that can be viewed directly in a browser without building.

## Key Files

- `index.html` - Homepage with club slider and about section
- `style.css` / `style.responsive.css` - Main stylesheet with Artisteer-generated CSS
- `script.js` / `script.responsive.js` - jQuery-based JavaScript (sliders, responsive handling, IE compatibility)
- `jquery.js` - jQuery library

## Common Tasks

### View the site locally
Open any `.html` file in a browser. Files are self-contained and don't require a server.

### Edit pages
Directly modify HTML files in `gsdcstl.org/`. The CSS is heavily inline and generated by Artisteer, making targeted edits possible but potentially fragile.

### Test responsive behavior
Resize your browser window while viewing pages; the JavaScript handles responsive layout changes via classes on `<html>` element.
